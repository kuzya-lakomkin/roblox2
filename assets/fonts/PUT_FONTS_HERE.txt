Сюда складывай шрифты (.ttf / .otf).

Имена берутся из client/asset_config.py. Шрифты разделены по элементам игры:

    assets/fonts/title.ttf   <- заголовки (SWAGA, названия экранов)
    assets/fonts/ui.ttf      <- кнопки и подписи меню/настроек
    assets/fonts/hud.ttf     <- боевой HUD (HP, очки, фаза)
    assets/fonts/chat.ttf    <- чат и подсказки
    assets/fonts/world.ttf   <- 3D-текст: ники над червями, шкала босса

Если какого-то файла нет — для этого элемента используется системный
шрифт по умолчанию (как сейчас). Можно положить только часть шрифтов.
Шрифт должен содержать кириллицу.
