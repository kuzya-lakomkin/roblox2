# Роблокс 2

Мультиплеерная 3D-игра на Python (Panda3D + свой authoritative-сервер на сокетах).
Вид от первого лица.

## Возможности
- 🌍 3D-арена с «зданиями», бег/прыжки
- 🐜 **Травля муравьёв** — стреляй секретной жидкостью по муравьям, колония возрождается через несколько секунд
- 🪱 **Флекс со шкибиди-червяком** — у каждого игрока питомец-червяк; эмоции `flex` / `dance` / `wave`
- 💬 **Чат** — общий чат для всех игроков
- 💦 **Обстрел игроков** «секретной жидкостью» — снаряды летят по дуге, попадание наносит урон
- 🏆 Очки за муравьёв и попадания, таблица HP/смертей в HUD

> Примечание: один из пунктов исходного описания (сексуальный контент) намеренно не реализован.

## Установка
```bash
pip install -r requirements.txt
```

## Запуск

1. Запусти сервер (одно окно терминала):
   ```bash
   python -m server.server
   ```

2. Запусти одного или нескольких клиентов (каждый в своём терминале):
   ```bash
   python -m client.main --name Стёпа
   python -m client.main --name Друг
   ```

Игра по сети: на сервере смотри его IP, на клиентах запускай
`python -m client.main --name Имя --host <IP-сервера>`
(порт по умолчанию 50007 должен быть открыт).

## Управление
| Клавиша | Действие |
|---------|----------|
| WASD | ходьба |
| Пробел | прыжок |
| Мышь | осмотр |
| ЛКМ | выстрел секретной жидкостью |
| F | флекс 💪 |
| G | танец 🕺 |
| V | помахать 👋 |
| Enter | открыть/отправить чат |
| Esc | закрыть чат / выйти |

## Структура
```
common/      общий код (конфиг, протокол сообщений)
server/      authoritative-сервер: симуляция мира, муравьёв, снарядов
client/      Panda3D-клиент: рендер, FPS-управление, сеть, UI
```

Сервер — авторитетный: он считает муравьёв, снаряды и попадания;
клиенты шлют своё положение и действия. Протокол — построчный JSON поверх TCP.
